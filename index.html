Mis Proyectos

Calculadora Básica

Aplicación web de calculadora desarrollada con HTML, CSS y JavaScript. Ofrece operaciones básicas y una interfaz intuitiva.

Calculadora Básica

Descripción: Esta calculadora básica permite realizar operaciones matemáticas fundamentales como suma, resta, multiplicación y división. Ha sido desarrollada utilizando HTML para la estructura, CSS para el estilo, y JavaScript para la funcionalidad.

Objetivos: Proporcionar una herramienta sencilla para realizar cálculos básicos en una interfaz amigable y accesible.

Tecnologías Utilizadas: HTML, CSS, JavaScript.

Características:

  • Interfaz intuitiva y fácil de usar.
  • Soporte para operaciones matemáticas básicas.
  • Diseño responsivo para adaptarse a diferentes dispositivos.

Desafíos: Asegurar que la interfaz sea clara y funcional en dispositivos de diferentes tamaños y solucionar problemas de redondeo en los cálculos.

Clima en Tu Lugar

Aplicación web que brinda información sobre el clima actual en tu ubicación.

Clima en tu Lugar

Descripción: Esta aplicación simple te brinda información sobre el clima actual en tu ubicación, mostrando la temperatura, presión atmosférica, humedad relativa y una descripción del clima actual.

Funcionalidades:

  • Muestra la temperatura actual.
  • Muestra la presión atmosférica.
  • Muestra la humedad relativa.
  • Proporciona una descripción del clima actual.

Requisitos:

  • Conexión a Internet para obtener los datos del clima.
  • Permiso del navegador para acceder a la ubicación del usuario.

API Utilizada: La aplicación utiliza la API de OpenWeatherMap para obtener la información del clima.

Tecnologías Utilizadas: HTML, CSS, JavaScript.

Blog de la localidad El Tigre

Proyecto colaborativo para crear un blog sobre la localidad El Tigre, destacando su historia y lugares de interés.

Blog de la localidad El Tigre

Descripción:Proyecto colaborativo que destaca la historia, cultura y lugares de interés de esta región. El objetivo es proporcionar a los residentes y visitantes una fuente de información detallada y atractiva sobre la localidad.

Objetivos: Crear una plataforma informativa sobre El Tigre. Promover el turismo y el conocimiento cultural de la región. Ofrecer información histórica y actualizada sobre eventos y lugares de interés.

Características:

  • Secciones dedicadas a la historia, cultura y eventos de El Tigre.
  • Galería de imágenes de lugares destacados de la localidad.
  • Integración de videos y contenido multimedia para una experiencia enriquecedora.

Participación: Este proyecto fue desarrollado en colaboración con dos compañeras, quienes contribuyeron significativamente en la recopilación de información, desarrollo de contenido y diseño del blog.

Tecnologías Utilizadas: HTML, CSS, JavaScript y Bootstrap

PetZone - Tienda de Mascotas

Proyecto de tienda de mascotas con frontend y backend. Utiliza Python, Flask y MySQL para el ABM de productos.

PetZone - Tienda de Mascotas

Descripción:Proyecto colaborativo de tienda de mascotas realizado en el curso "Codo a Codo". Incluye desarrollo frontend y backend utilizando Python con Flask y MySQL para gestionar el ABM de productos.

Objetivos:

  • Crear una plataforma de gestión de productos para una tienda de mascotas.
  • Facilitar la administración y actualización de inventarios.
  • Proporcionar una interfaz amigable para los usuarios.

Características:

  • Gestión de productos (alta, baja, modificación).
  • Integración con base de datos MySQL.
  • Diseño responsivo con Bootstrap.
  • Interfaz intuitiva para facilitar la navegación y administración.

Participación:Este proyecto fue desarrollado en colaboración con dos compañeras en el curso "Codo a Codo", quienes contribuyeron significativamente en la implementación de funcionalidades y el diseño del sistema.

Tecnologías Utilizadas:HTML, CSS, JavaScript, Bootstrap, Python, Flask, MySQL.
